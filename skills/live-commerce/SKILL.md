## 直播策划SOP

### 1. 预热阶段（开播前3-7天）

**内容预热：**
- [ ] 发布预热短视频：剧透福利品、爆款预告、主播剧透
- [ ] 短视频挂直播预约组件
- [ ] 社群/企微推送预约提醒
- [ ] 直播间标题+封面图优化

**预热短视频模版：**
```
标题： [价格/福利] + [产品] + [时间]
内容结构：
- 前3秒：价格悬念或痛点切入
- 中间：产品亮点展示
- 结尾：引导预约「今晚8点直播间见」
```

**用户触达节奏：**
| 时间节点 | 动作 |
|---------|------|
| 开播前3天 | 发布预告短视频 |
| 开播前1天 | 社群/企微推送预告海报 |
| 开播前2小时 | 开播提醒（系统+手动） |
| 开播前15分钟 | 社群倒计时+预告福利 |

### 2. 开播阶段（开场黄金3分钟）

**开场节奏：**
1. **欢迎+自我介绍**（15秒）：建立信任
2. **本场福利预告**（30秒）：留住用户的核心钩子
3. **抽奖/互动引导**（30秒）：点赞、关注、评论指定话术
4. **第一个福利品上架**（45秒）：最低价的钩子品，制造抢购氛围

**开场话术框架：**
```
欢迎新来的朋友！今天这场直播给大家准备了[数量]款福利品。
先点个关注，主播每[时间]抽一轮大奖。
今天第一款福利，市场价[X]元，直播间只要[Y]元，倒计时3-2-1上链接！
```

### 3. 互动阶段（持续加热）

**互动策略：**
| 手段 | 频率 | 目的 |
|------|------|------|
| 点赞到X万送福利 | 每15-20分钟 | 停留时长 |
| 评论指定关键词抽奖 | 每10-15分钟 | 互动率 |
| 关注送优惠券 | 整场持续 | 涨粉 |
| 分享直播间领奖品 | 每30分钟 | 裂变拉新 |

**直播间停留话术：**
```
刚进来的朋友不要走，今天这款是品牌方的首发福利，只有直播间才有这个价格。
左上角领个优惠券再下单，更划算。
```

### 4. 转化阶段（核心出货）

**产品讲解节奏（单品循环）：**
1. **痛点引入**（15秒）：这个产品解决什么问题
2. **卖点展示**（30-60秒）：现场演示/试穿/试用
3. **价格锚点**（15秒）：市场价 vs 直播价对比
4. **限时优惠**（15秒）：倒计时/限量营造紧迫感
5. **逼单话术**（15秒）：下单催促，打消犹豫
6. **上链接**（5秒）：3-2-1倒计时

**逼单话术库：**
````
场景：用户犹豫
「这款库存只剩最后XX单，拍完就没有了」
「今天下单额外送XX赠品，错过今天就没有了」
「这个价格我们跟品牌方申请了一个月，今天破价卖」
「已经拍了XX单了，还没拍的朋友抓紧」

场景：价格质疑
「同款旗舰店卖XXX，今天直播间立省XXX」
「品质一模一样，只是我们直播间补贴」
「支持全网比价，买贵退差价」
````

### 5. 复盘阶段（直播后24小时内）

**数据复盘维度：**
| 指标 | 计算方式 | 行业参考 |
|------|---------|---------|
| 观看人次（UV） | 进入直播间总人数 | 品牌自播500-5000/场 |
| 最高在线（PCU） | 同时在线峰值 | 占UV 5-15% |
| 平均停留时长 | 总时长/UV | >2分钟为合格 |
| 互动率 | (点赞+评论+分享)/UV | >5%为合格 |
| 转化率 | 下单人数/UV | 1-3%为正常，5%+优秀 |
| 客单价 | GMV/订单数 | 视品类而定 |
| 退款率 | 退款金额/GMV | <10%为健康 |

**复盘问题清单：**
- 开场留住率如何？哪个环节流失最多？
- 哪款产品的转化最好？原因是什么？
- 话术哪里卡壳了？下次怎么优化？
- 流量来源分布？自然推荐vs付费占比？
- 下次直播要延续什么？改进什么？

## 主播培训与话术设计

### 主播能力模型

| 能力维度 | 具体要求 | 训练方法 |
|---------|---------|---------|
| 产品知识 | 熟悉每款产品的卖点和用户场景 | 每日产品培训会 |
| 表达能力 | 清晰、有感染力、不卡壳 | 话术背稿+模拟直播 |
| 互动能力 | 实时回复评论、引导互动 | 看播训练+老主播带教 |
| 应变能力 | 处理突发状况（技术故障、差评） | 应急预案演练 |
| 数据敏感度 | 根据实时数据调整节奏 | 数据看板培训 |

### 话术设计原则

1. **开场30秒决定留存**：第一句话必须有钩子
2. **价格对比是核心武器**：永远让用户感受到"赚到了"
3. **从众心理驱动下单**："已经XX人下单了"
4. **稀缺性制造紧迫感**：限量、限时、限名额
5. **信任感是转化的基础**：真实演示>口播

### 不同场景话术模板

**新品首发：**
```
今天这款是我们跟品牌方联合首发的XXX。
市面上还买不到，只有直播间有现货。
首批只有XXX件，拍完等预售。
```

**清仓特卖：**
```
这批是工厂尾单，跟正品品质一样，没有包装所以便宜。
只有XXX件，码数不全了，拍完断货不补。
```

**节日专场：**
```
[节日]快到了，送朋友/送家人/送自己……
今天下单顺丰包邮，[节日前]保证到货。
限时[节日]专属价，错过等一年。
```

## 直播间转化优化

### 转化路径设计

```
进入直播间 → 停留 → 互动（点赞/评论） → 产品种草 → 领取优惠券 → 下单 → 支付成功 → 加企微
```

每个环节都有对应的运营动作和话术配合。核心原则：**减少用户决策成本，降低下单门槛。**

### 限时优惠策略

| 优惠类型 | 玩法 | 效果 |
|---------|------|------|
| 限时秒杀 | 整点秒杀，限量X件 | 制造集中抢购氛围 |
| 阶梯优惠 | 前10分钟下单享额外折扣 | 加速首单决策 |
| 满减叠加 | 满X减Y，可叠加优惠券 | 提升客单价 |
| 赠品驱动 | 下单送价值XX赠品 | 提高转化率 |
| 福袋抽奖 | 下单用户参与抽奖 | 提升下单动力 |

### 专属福利设计

**直播专享价：**
```
日常价：XXX元
直播间价：XXX元
立省：XXX元
```

**直播间专享赠品：**
```
日常购买：无赠品
直播间下单：送价值XXX的XX礼包
仅限今日前XXX名
```

**粉丝专享券：**
- 关注领券：5-10元无门槛
- 粉丝群专享：满减券
- 会员专享：折上折

### 提高客单价技巧

1. **搭配推荐**：买了A的还买了B
2. **凑单满减**："再加XX元即可享受满减"
3. **升级款推荐**：标准版→Pro版
4. **多件优惠**："买2件打8折，买3件打7折"
5. **加价购**："+XX元换购价值XXX的产品"

## 私域专场直播

私域直播面向已有用户关系（企微好友、社群成员），转化率和客单价通常高于公域直播。

### 私域直播特点

| 维度 | 公域直播 | 私域直播 |
|------|---------|---------|
| 流量来源 | 推荐算法 | 社群+企微+公众号 |
| 用户关系 | 陌生人 | 已有信任基础 |
| 转化率 | 1-3% | 5-15% |
| 客单价 | 相对低 | 相对高 |
| 退货率 | 较高 | 较低 |
| 核心目标 | 拉新+成交 | 复购+客单价 |

### 私域专场SOP

**预热期（开播前3-5天）：**
1. 企微标签筛选目标用户（近30天购买、高活跃等）
2. 企微1对1发送专属预约海报
3. 社群推送预告+预热福利
4. 公众号推文联动

**直播期：**
1. 企微专属暗号：回复特定关键词领额外优惠
2. 社群实时播报：「刚刚上了XX爆款，抢疯了」
3. 老客专享价：比公域低5-10%
4. 群内互动抽奖

**复盘期：**
1. 企微回访未下单用户
2. 已购用户社群晒单有奖
3. 沉淀到企业微信做长期触达

### 私域直播话术

```
今天是咱们社群专场的福利日，外面的朋友看不到这个价格。
只有添加了企微好友的家人们才能享受。
评论区扣「专属福利」领额外的优惠券。
```

## 品牌自播 vs 达人直播

2026年品牌自播占比已超过达人直播，成为主流选择。

### 对比分析

| 维度 | 品牌自播 | 达人直播 |
|------|---------|---------|
| 流量稳定性 | 依赖运营能力，需持续投入 | 达人自带流量 |
| 内容把控 | 完全自主 | 受限于达人风格 |
| 利润率 | 较高（无坑位费） | 较低（坑位费+佣金） |
| 长期价值 | 品牌资产沉淀 | 单次曝光为主 |
| 运营成本 | 较高（团队+场地+投流） | 较低（选品+对接） |
| 用户信任 | 需从零建立 | 达人信用背书 |
| 退货率 | 可控 | 较高（冲动消费） |

### 品牌自播成熟度模型

| 阶段 | 特征 | 关键动作 |
|------|------|---------|
| 试水期（0-3月） | 日均GMV<1万 | 测品、找话术、跑通流程 |
| 成长期（3-6月） | 日均GMV 1-5万 | 稳定开播、优化转化、投流测试 |
| 成熟期（6-12月） | 日均GMV 5-20万 | 矩阵号运营、建立直播团队 |
| 规模化（12月+） | 日均GMV 20万+ | 多平台覆盖、品牌IP化 |

### 达人合作策略

**选号标准：**
- 粉丝量不是唯一标准，看互动率和粉丝画像匹配度
- 近30天场均观看和GMV趋势
- 评论区真实度（排除数据造假）
- 与品牌调性是否一致

**合作方式：**
| 方式 | 适合品牌阶段 | 风险 |
|------|------------|------|
| 纯佣合作 | 成熟品牌，强产品力 | 达人配合度低 |
| 坑位费+佣金 | 新品牌冲量 | 可能亏损 |
| 保量协议 | 大促冲销量 | 需对赌 |
| 年框合作 | 长期品牌建设 | 锁定费用 |

## AI数字人主播趋势

AI数字人直播在2026年进入规模化应用阶段，尤其适合非黄金时段承接流量。

### 适用场景

| 场景 | 推荐度 | 说明 |
|------|--------|------|
| 深夜时段（0-8点） | 高 | 真人成本高，数字人承接自然流量 |
| 日播长尾时段 | 高 | 非核心时段填充 |
| 商品展示型直播 | 中 | 话术固定，不需实时互动 |
| 复杂讲解型直播 | 低 | 数字人应变能力不足 |
| 高互动要求专场 | 极低 | 推荐真人主播 |

### 数字人直播关键要素

1. **形象选择**：2D真人克隆 > 3D虚拟形象 > 卡通形象
2. **话术录制**：录制真人话术片段，AI拼接循环
3. **互动兜底**：预设常见问题回复，复杂问题转人工
4. **合规要求**：平台对数字人直播有标注要求（需标注"AI直播"）
5. **转化效果**：通常为真人直播的30-60%，但边际成本极低

### 主流数字人平台

| 平台 | 特点 | 价格区间 |
|------|------|---------|
| 硅基智能 | 2D真人克隆成熟 | 1-5万/年 |
| 腾讯智影 | 视频号生态配合好 | 按量计费 |
| 商汤如影 | 3D形象效果好 | 2-10万/年 |
| 腾讯元宝 | 接入大模型能力强 | 内测中 |

## 数据监测与优化

### 实时数据看板

| 指标 | 监测频率 | 告警阈值 |
|------|---------|---------|
| 实时在线人数 | 实时 | 低于预期50%告警 |
| 商品点击率 | 每品 | <5%优化话术 |
| 下单转化率 | 每分钟 | 低于均值30%检查 |
| 退款率 | 每30分钟 | >15%检查商品质量 |
| 投放ROI | 每10分钟 | <1.2暂停投放 |

### A/B测试方向

- **话术测试**：同一产品的不同介绍方式
- **价格测试**：不同折扣力度的转化差异
- **福利品测试**：不同钩子品的吸引力
- **直播间装修测试**：背景、灯光、陈列
- **投流素材测试**：不同创意的引流效果

### 常用工具

- **数据分析**：蝉妈妈、飞瓜数据、抖查查
- **直播间管理**：巨量百应、直播中控台
- **投放工具**：巨量千川、腾讯广告
- **私域工具**：企业微信、微盟、有赞
