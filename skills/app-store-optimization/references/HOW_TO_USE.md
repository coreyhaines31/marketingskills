# How to Use the App Store Optimization Skill

## Overview

This skill provides comprehensive ASO capabilities for optimizing your app's presence on the Apple App Store and Google Play Store. Below are recommended workflows based on your situation.

## Workflows

### New App Launch

1. **Keyword Research** — Start with `keyword_analyzer.py` to identify high-value keywords
2. **Competitor Analysis** — Use `competitor_analyzer.py` to benchmark against rivals in your category
3. **Metadata Optimization** — Apply findings with `metadata_optimizer.py` for title, description, and keyword field
4. **Launch Checklist** — Run through `launch_checklist.py` to ensure nothing is missed
5. **A/B Test Plan** — Set up post-launch tests with `ab_test_planner.py`

### Existing App Optimization

1. **ASO Scoring** — Start with `aso_scorer.py` to identify your weakest areas (score < 60 = high priority)
2. **Competitor Gap Analysis** — Use `competitor_analyzer.py` to find keyword and rating gaps
3. **Metadata Update** — Refresh metadata using `metadata_optimizer.py`
4. **Review Analysis** — Mine insights from `review_analyzer.py`

### International Growth

1. **Market Identification** — Use `localization_helper.py` to identify priority markets by ROI
2. **ROI Calculation** — Calculate payback period before committing budget
3. **Keyword Adaptation** — Adapt (not just translate) keywords for each market
4. **Phased Rollout** — Follow the phased implementation plan generated by the helper

### Ongoing Maintenance

- **Monthly**: Track keyword rankings, review new reviews, update A/B tests
- **Quarterly**: Refresh metadata, run competitor analysis, review ASO score
- **With each update**: Update "What's New" section, respond to recent reviews

## What to Provide

The more context you give, the better the output:

- **App category** — Affects keyword competition and benchmarks
- **Target audience** — Shapes description copy and keyword priorities
- **Current metrics** — Ratings, download counts, conversion rate
- **Competitor names** — Enables direct competitive benchmarking
- **Platform(s)** — Apple, Google Play, or both

## Scope Boundaries

This skill covers:
- Apple App Store and Google Play Store
- Organic ASO (not paid user acquisition)
- Consumer and B2C apps

This skill does NOT cover:
- Web apps or PWAs
- Enterprise app distribution
- Paid advertising (see `paid-ads` skill)
- App development or UI/UX
- Technical submission issues
