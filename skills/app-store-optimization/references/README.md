# App Store Optimization (ASO) Skill

## Overview

Complete App Store Optimization toolkit for researching, optimizing, and tracking mobile app performance on Apple App Store and Google Play Store.

## Capabilities

| Area | Description |
|------|-------------|
| Research & Analysis | Keyword research, competitor benchmarking, market trends |
| Metadata Optimization | Title, description, keyword field with character limit validation |
| Conversion Optimization | A/B testing frameworks, visual asset optimization |
| Rating & Review Management | Sentiment analysis, response strategies, rating improvement |
| Launch & Update Strategies | Pre-launch checklists, timing optimization, update cadence |
| Analytics & Tracking | ASO health scoring (0-100), keyword ranking, conversion metrics |
| Localization | Multi-market strategy, ROI calculation, keyword adaptation |

## Scripts

| Script | Purpose |
|--------|---------|
| `keyword_analyzer.py` | Keyword research, scoring, and long-tail discovery |
| `metadata_optimizer.py` | Title, description, and keyword field optimization |
| `competitor_analyzer.py` | Competitive analysis and gap identification |
| `aso_scorer.py` | ASO health score (0-100) across four dimensions |
| `ab_test_planner.py` | A/B test design, sample sizing, and significance calculation |
| `localization_helper.py` | Market prioritization, ROI calculation, keyword adaptation |
| `review_analyzer.py` | Sentiment analysis, issue detection, feature request extraction |
| `launch_checklist.py` | Pre-launch checklist generation and timing optimization |

## Platform Character Limits

### Apple App Store
| Field | Limit |
|-------|-------|
| Title | 30 chars |
| Subtitle | 30 chars |
| Promotional Text | 170 chars |
| Description | 4,000 chars |
| Keywords | 100 chars |
| What's New | 4,000 chars |

### Google Play Store
| Field | Limit |
|-------|-------|
| Title | 50 chars |
| Short Description | 80 chars |
| Full Description | 4,000 chars |
