# SEO Content Checklist

Complete checklist for on-page SEO and content optimization. Review before publishing.

## Keyword Optimization

- [ ] Primary keyword identified and researched
- [ ] Secondary/supporting keywords identified (3-5)
- [ ] Search intent matches content type (informational, commercial, transactional)
- [ ] Long-tail variations included naturally
- [ ] Keyword difficulty assessed and realistic for domain authority

## Title Tag

- [ ] Contains primary keyword (preferably near beginning)
- [ ] Length: 50-60 characters
- [ ] Compelling and click-worthy
- [ ] Unique (not duplicated across site)
- [ ] Accurately describes page content
- [ ] Brand name included (if space permits, at end)

## Meta Description

- [ ] Length: 150-160 characters
- [ ] Contains primary keyword naturally
- [ ] Includes clear value proposition
- [ ] Has call-to-action or entices click
- [ ] Unique for each page
- [ ] Accurately summarizes content

## URL Structure

- [ ] Short and descriptive
- [ ] Contains primary keyword
- [ ] Uses hyphens between words (not underscores)
- [ ] Lowercase letters only
- [ ] No special characters or parameters
- [ ] No stop words unless necessary for clarity

## Header Structure (H1-H6)

- [ ] Single H1 tag containing primary keyword
- [ ] H2s used for major sections
- [ ] H3s used for subsections
- [ ] Logical hierarchy maintained (no skipping levels)
- [ ] Keywords distributed naturally in headers
- [ ] Headers are descriptive and useful for readers

## Content Quality

### Opening Section
- [ ] Main question/query answered in first 100 words
- [ ] Hook engages reader immediately
- [ ] Primary keyword appears in first paragraph
- [ ] Clear preview of what content covers

### Body Content
- [ ] Comprehensive coverage of topic
- [ ] Answer-first paragraph structure throughout
- [ ] Appropriate length for topic (typically 1,500-2,500 words for competitive terms)
- [ ] Unique, original content (not duplicated)
- [ ] Scannable with short paragraphs (2-4 sentences)
- [ ] Varied sentence structure and length
- [ ] Active voice preferred
- [ ] No keyword stuffing (natural density)

### E-E-A-T Signals
- [ ] First-hand experience demonstrated
- [ ] Expert credentials/qualifications shown
- [ ] Citations from authoritative sources included
- [ ] Author byline with bio/credentials
- [ ] Publication and update dates displayed
- [ ] Transparent about sources and methodology
- [ ] Balanced perspective on controversial topics

### Factual Accuracy
- [ ] All facts verified against reliable sources
- [ ] Statistics include source citations
- [ ] No outdated information
- [ ] Expert quotes attributed correctly
- [ ] No unsubstantiated claims

## Internal Linking

- [ ] Links to relevant pillar/cornerstone content
- [ ] Descriptive anchor text (not "click here")
- [ ] 3-5 internal links minimum for long-form content
- [ ] Links to related blog posts/pages
- [ ] No broken internal links
- [ ] Link to category/topic cluster pages

## External Linking

- [ ] Links to authoritative, reputable sources
- [ ] 2-3 external links minimum
- [ ] Links open in new tab (where appropriate)
- [ ] No links to competitors' commercial pages
- [ ] Source links for statistics and claims
- [ ] Check that linked pages are still live

## Images and Media

- [ ] Relevant images included
- [ ] Descriptive alt text with keywords (where natural)
- [ ] Images compressed for fast loading
- [ ] Appropriate file names (descriptive, hyphenated)
- [ ] Captions included where helpful
- [ ] Original or properly licensed images
- [ ] Featured image selected

## FAQ Section (For AEO)

- [ ] 3-5 questions included
- [ ] Questions match real search queries
- [ ] Answers are concise and direct
- [ ] Question format in H3 or H4 headers
- [ ] Answers include relevant keywords
- [ ] Schema markup added for FAQ section

## Featured Snippet Optimization

- [ ] Target snippet type identified (paragraph, list, table)
- [ ] Content formatted to match snippet type
- [ ] Direct answer in 40-60 words for paragraph snippets
- [ ] Clear numbered/bulleted lists for list snippets
- [ ] Properly formatted tables for comparison snippets

## Technical SEO

- [ ] Page loads in under 3 seconds
- [ ] Mobile-responsive design verified
- [ ] No broken links
- [ ] Canonical URL set correctly
- [ ] Schema markup implemented and validated
- [ ] No duplicate content issues
- [ ] Page is indexable (no noindex tag unless intentional)

## Schema Markup

- [ ] Appropriate schema type selected
- [ ] JSON-LD format used (Google recommended)
- [ ] Schema validated with Google Rich Results Test
- [ ] Required properties included
- [ ] No schema errors or warnings
- [ ] Schema matches visible page content

## GEO Optimization (For AI Citations)

- [ ] Facts stated explicitly, not implied
- [ ] Statistics included with sources
- [ ] Expert quotes with attribution
- [ ] Clear, fluent language throughout
- [ ] Self-contained, quotable statements
- [ ] Content is easily extractable by AI
- [ ] Unique insights or data included

## Pre-Publish Final Check

- [ ] Grammar and spelling checked
- [ ] Read aloud for flow and clarity
- [ ] All links tested and working
- [ ] Preview on mobile device
- [ ] Schema tested with Google Rich Results Test
- [ ] Meta data complete and optimized
- [ ] Publication date set correctly
- [ ] Author assigned with proper bio
