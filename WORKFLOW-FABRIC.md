# 面料经销商国内市场营销工作流

## 📋 你的情况

- **行业**: 面料经销
- **目标客户**: 服装厂、电商卖家、设计师品牌
- **目标**: 通过国内市场营销获取更多客户

---

## 🚀 完整工作流（4周计划）

### 第1周：搭建基础（私域流量）

#### Step 1: 配置产品上下文
```bash
# 进入项目目录
cd /home/six/YifuAIForge/marketingskills-cn

# 复制产品上下文模板
cp .agents/product-marketing-cn.md .agents/product-marketing.md

# 编辑填写你的产品信息
# 重点填写：
# - 产品/服务描述：面料经销、面料批发
# - 目标客户：服装厂、电商卖家、设计师品牌
# - 核心价值：品种齐全、价格优势、快速交货
# - 主要卖点：源头厂家直供、支持小批量、免费寄样
```

#### Step 2: 启用企业微信
```bash
# 使用 wechat-ecosystem 技能
# 触发词：企业微信、私域运营、客户管理

# 执行步骤：
1. 注册企业微信（免费）
2. 创建员工账号
3. 设置自动欢迎语："您好！我是XX面料的小王，专注面料批发10年，有什么可以帮您？"
4. 设置客户标签：服装厂、电商卖家、设计师、贸易公司
5. 准备朋友圈内容：每天发1-2条面料新品或案例
```

#### Step 3: 搭建小程序商城
```bash
# 使用 mini-program-ecommerce 技能
# 触发词：小程序、有赞、微盟

# 推荐方案（低成本）：
1. 注册有赞微商城（基础版约3000元/年）
2. 上传面料产品（按品类分类：棉、麻、丝绸、化纤等）
3. 设置价格体系：批发价、零售价、VIP价
4. 配置物流：对接快递100、顺丰
5. 生成小程序码，打印到名片和包装上
```

---

### 第2周：内容营销（获取流量）

#### Step 4: 开通抖音账号
```bash
# 使用 short-video-marketing 技能
# 触发词：抖音、短视频、内容创作

# 账号定位：
- 名称：XX面料小王（带人设）
- 简介：面料批发 | 10年经验 | 免费寄样 | 服装厂合作
- 头像：真人照片（增加信任感）

# 内容规划（每天1条）：
周一：面料知识科普（什么是天丝？）
周二：面料生产过程（工厂实拍）
周三：客户案例展示（用我们的面料做的衣服）
周四：面料价格对比（为什么我们便宜）
周五：行业热点（流行趋势）
周六：客户好评/订单展示
周日：生活化内容（工作日常）
```

#### Step 5: 开通小红书账号
```bash
# 使用 xiaohongshu-marketing 技能
# 触发词：小红书、种草、笔记

# 账号定位：
- 名称：面料采购指南
- 简介：帮助服装厂找到好面料 | 每天分享面料知识
- 定位：专业+干货

# 内容规划：
- 面料选购指南
- 面料避坑攻略
- 面料价格行情
- 客户合作案例
- 面料趋势分析
```

### 第3周：社群运营（深度转化）

#### Step 6: 建立客户社群
```bash
# 使用 community-operations 技能
# 触发词：社群、微信群、用户运营

# 社群规划：
1. 建立"面料采购交流群"（微信群）
2. 入群门槛：必须是服装厂/电商卖家
3. 群规则：
   - 禁止发广告
   - 每周一次面料知识分享
   - 新品优先在群里发布
   - 提供1对1选料服务

4. 社群活动：
   - 每周三：面料知识直播
   - 每周五：新品发布
   - 每月底：清仓特惠
```

#### Step 7: KOL合作
```bash
# 使用 xiaohongshu-marketing 技能
# 触发词：KOL、达人合作、种草

# 合作策略：
1. 找服装类KOL（粉丝1-10万）
2. 提供免费面料样品
3. 请他们发布"面料测评"笔记
4. 评论区引导加微信咨询
```

### 第4周：广告投放（放大流量）

```bash
# 触发词：腾讯广告、朋友圈广告
```

```bash

# 触发词：巨量引擎、抖音广告

# 广告计划：
- 预算：1000-3000元/测试
- 定向：服装行业、电商行业
- 素材：短视频（面料展示+价格优势）
- 落地页：表单收集（姓名+电话+需求）
- 转化目标：收集销售线索
```

## ⚠️ 注意事项

1. **不要急着投广告**：先做好内容和私域，有基础再放大
2. **坚持内容创作**：每天发内容比一次性投广告更重要
3. **重视客户服务**：面料行业口碑很重要
4. **数据驱动**：每周分析数据，及时调整策略
5. **控制成本**：初期小投入测试，有效果再加大
