# G2

Plataforma de avaliações de software e pesquisa para compradores B2B. Acesse avaliações, dados de produtos, comparações com concorrentes e sinais de intenção de compra.

## Capacidades

| Integração | Disponível | Observações |
|-------------|-----------|-------|
| API | ✓ | Reviews, Products, Reports, Categories, Tracking |
| MCP | - | Não disponível |
| CLI | ✓ | [g2.js](../clis/g2.js) |
| SDK | - | REST API com formato JSON:API |

## Autenticação

- **Tipo**: API Token
- **Header**: `Authorization: Token token={YOUR_API_TOKEN}`
- **Content-Type**: `application/vnd.api+json` (JSON:API)
- **Obter token**: G2 Admin Portal > Integrations > API Tokens
- **Docs**: https://data.g2.com/api/docs

## Operações comuns de agent

### Listar reviews (respostas de pesquisa)

```bash
GET https://data.g2.com/api/v1/survey-responses?page[size]=25&page[number]=1

Headers:
  Authorization: Token token={API_TOKEN}
  Content-Type: application/vnd.api+json
```

### Buscar uma review específica

```bash
GET https://data.g2.com/api/v1/survey-responses/{id}

Headers:
  Authorization: Token token={API_TOKEN}
  Content-Type: application/vnd.api+json
```

### Filtrar reviews por produto

```bash
GET https://data.g2.com/api/v1/survey-responses?filter[product_id]={product_id}&page[size]=25

Headers:
  Authorization: Token token={API_TOKEN}
  Content-Type: application/vnd.api+json
```

## Padrão da API

O G2 segue a especificação JSON:API (https://jsonapi.org/):
- As respostas usam a estrutura `data`, `attributes`, `relationships`, `meta`
- Paginação: parâmetros de consulta `page[number]` e `page[size]`
- Filtros: parâmetros de consulta `filter[field]=value`
- As reviews são retornadas por padrão da mais recente para a mais antiga (10 por página por padrão)

## Métricas principais

### Métricas de review
- `star_rating` - Classificação geral em estrelas
- `title` - Título da review
- `comment_answers` - Respostas estruturadas da review (pontos positivos, negativos e recomendações)
- `submitted_at` - Data de envio da review
- `is_public` - Se a review está visível publicamente

### Métricas de produto
- `name` - Nome do produto
- `slug` - URL slug de URL no G2
- `avg_rating` - Média de classificação em estrelas
- `total_reviews` - Total de reviews
- `category` - Categoria no G2

### Buyer Intent (Tracking)
- `company_name` - Nome da empresa visitante
- `page_visited` - URL da página do G2 visitada
- `visited_at` - Timestamp da visita
- `activity_type` - Tipo de atividade do comprador

## Parâmetros

### Paginação
- `page[number]` - Número da página (padrão: 1)
- `page[size]` - Itens por página (padrão: 10, máx: 100)

### Filtros de review
- `filter[product_id]` - Filtrar por ID do produto
- `filter[state]` - Filtrar por status da review

### Filtros de tracking
- `filter[start_date]` - Data inicial (YYYY-MM-DD)
- `filter[end_date]` - Data final (YYYY-MM-DD)

## Quando usar

- Monitorar e analisar reviews de produtos de software
- Acompanhar sinais de buyer intent de visitantes do G2
- Coletar dados de comparação com concorrentes para posicionamento
- Alimentar CRM ou automação de marketing com dados de reviews
- Criar conteúdo de social proof a partir de reviews do G2
- Acompanhar ranking em categorias e presença em relatórios do G2

## Limites de taxa

- 10.000 requisições por hora por API token
- Implemente exponential backoff em respostas 429
- Faça cache de resultados quando possível para reduzir chamadas à API
